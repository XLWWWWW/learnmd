#### awk取列

- -F 指定列分割符号，（awk的默认的分隔符为空格 连续的空格 tab）

- $0  表示取出整行的内容
- $NF 表示最后一列
- 取列的时候 一般都是 `{print $xx}` 的形式

## sort 

排序，关键字

| 关键字        | 描述                                                   |
| ------------- | ------------------------------------------------------ |
| -r            | 逆序 默认为升序 逆序为降序                             |
| -n            | 按照数值的大小排序                                     |
| -k            | 按照指定的列进行排序                                   |
| -b            | 忽略每行前面开始出的空格字符。从第一个可见字符开始比较 |
| -t <分隔字符> | 指定排序时所用的栏位分隔字符。                         |
| -o <输出文件> | <输出文件> 将排序后的结果存入指定的文件。              |

> 这里 写一个高德地图的例子
>
> ```shell
> // access.log
> http://www.baidu.com
> http://www.taobao.com
> http://www.qq.com
> http://www.taobao.com
> 
> # 统计各个网址出现的频率 并使用逆序排序 形式为 http://www.taobao.com:2
> ```
>
> 
>
> ```bash
> cat access.log|awk '{count[$1]++}END{for(i in count)print i":"count[i]}' |sort -t ":" -rnk2
> ```
>
> 就是使用 awk数组进行排序统计 这里需要注意的就是 
>
> - 我们print 添加冒号时 需要“:"
> -  另外 sort 使用-t 设置分割符
